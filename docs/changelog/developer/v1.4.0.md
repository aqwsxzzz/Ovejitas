# v1.4.0 Developer Changelog

## Scope
- Release tag: v1.4.0
- Baseline: v1.3.0
- Primary delivery areas: ledger-aligned workflows, finance dashboard rebuild, inventory material flows, livestock event/feeding UX, navigation/auth copy cleanup.
- Source PRs: #41 (feature bundle), #42 (development to main release merge).

## Architecture / Design Changes
- Introduced a new finance dashboard composition centered on `FinanceDashboardShell` and focused finance subcomponents.
- Expanded feature-level modularization for inventory material timeline/forms/panels.
- Continued migration toward contextual action entry points (removal of global quick action tab in bottom nav).
- Reinforced ledger-first flow direction through updated docs/prompts and feature wiring.

## API and Data Contract Changes
- Livestock and flock query/mutation surfaces expanded for ledger-backed event and asset operations.
- Reports and finance adapters updated to support richer dashboard and profitability outputs.
- Material flow handling now includes structured purchase/sale/consumption forms and timeline contracts.
- No breaking transport-layer migration script required in this release; behavior changes are additive at FE integration level.
